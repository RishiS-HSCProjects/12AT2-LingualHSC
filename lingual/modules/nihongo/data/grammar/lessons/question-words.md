# Using か (ka)

- どこ + か → somewhere
- だれ + か → someone
- 何 + か → something
- いつ + か → sometime

/w Do not use を or は when using this structure. You can use other particles including (but not limitted to) が, に, へ, の, etc. \

Examples:

- じゃ、私[わたし]達[たち]しか知[し]らないどこかに行かないか → So, why don't we go somewhere only we know?
- だれかと話したい → I want to talk to someone.

# Using も (mo)

どこも → Everywhere or nowhere

- どこ**も**忙[いそが]しい → It's busy everywhere.
- どこ**も**忙[いそが]しくない → It's not busy anywhere.

だれも → Everybody or nobody

- だれ**も**好きだ → I like everyone
- だれ**も**好きじゃない → I dont like anyone

いつも → Always or never

- いつ**も**行く
- いつ**も**行かない


何も → Everything or nothing

- 何**も**食べたい → I want to eat everything.
- 何**も**食べたくない → I don't want to eat everything.

Examples:

- 今だれもと話したくない → I don't want to talk to anyone right now

# Using でも (demo)

- どこでも → Anywhere
- だれでも → Anyone
- 何[なん]でも → Anything
- いつでも → Anytime

Examples:
- 何でもいい → Anything is fine
- いつでもいい → Anything is fine
