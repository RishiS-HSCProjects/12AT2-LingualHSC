Question words can change meaning when combined with particles. In this lesson, we focus on how か, も, and でも change words like どこ, だれ, 何, and いつ.

# Using か (ka)

Use か after a question word to mean an unspecified person, thing, place, or time.

:::blockquote Question word + か :::

- どこ + か → somewhere
- だれ + か → someone
- 何 + か → something
- いつ + か → sometime

/w Do not use を or は when using this structure. You can use other particles including (but not limited to) が, に, へ, の, etc. \
/i Think of か here as "some/any one", where the exact detail is unknown. \

Examples:

- じゃ、私[わたし]達[たち]だけが知[し]っているどこかに行かないか → So, why don't we go somewhere only we know?
- だれかと話[はな]したい → I want to talk to someone.
- 何[なに]か食[た]べたい → I want to eat something.
- いつか日本[にほん]に行きたい → I want to go to Japan someday.

# Using も (mo)

With question words, も changes meaning based on polarity:

- Positive sentence → every/all
- Negative sentence → no/not any

/t This is why question word + も can mean two opposite things depending on whether the verb is positive or negative. \
/i When another particle is needed, も attaches to it (e.g. にも, へも, でも, とも). \

## どこも → Everywhere or nowhere

- どこ**も**にぎやかです → It's lively everywhere.
- 今日[きょう]はどこ**も**開[あ]いていません → Nowhere is open today.

## だれも → Everybody or nobody

- クラスではだれ**も**漢字[かんじ]が読[よ]めます → Everyone in the class can read kanji.
- 今日はだれ**も**来ません → No one is coming today.

## いつも → Always

- 私[わたし]はいつ**も**早[はや]く起[お]きます → I always wake up early.
- 私はいつ**も**コーヒーを飲[の]むわけではありません → I don't always drink coffee.


## 何も → Everything or nothing

- 何**も**食べたい → I want to eat everything.
- 何**も**食べたくない → I don't want to eat anything *OR* I want to eat nothing.

Examples:

- 今[いま]だれとも話[はな]したくない → I don't want to talk to anyone right now.
- 休[やす]みの日[ひ]はどこにも行きません → On holidays, I don't go anywhere.

# Using でも (demo)

Use でも after a question word to express open choice: "any ... is fine."

It is similar to saying "**even if it's...**" in English, acting as a noun in the [ても structure](grammar:te-mo).

:::blockquote Question word + でも :::

- どこでも → Anywhere
- だれでも → Anyone
- 何[なん]でも → Anything
- いつでも → Anytime

/i でも has a free-choice nuance: any option is acceptable. \

Examples:
- 何でもいい → Anything is fine.
- いつでもいい → Anytime is fine.
- だれでも参[さん]加[か]できます → Anyone can participate.
- どこでも勉[べん]強[きょう]できます → You can study anywhere.

## Quick Comparison

- Question word + か: some/unknown one
- Question word + も: every/all (positive) or none/not any (negative)
- Question word + でも: any/free choice
